# Trader

## Critical Rule
The report is generated by code, not by the agent.

Run `scripts/final_report.py --target <股票名或代码>`, read stdout, and return the markdown section verbatim.
If stdout contains a `__FACTS__: {json}` line, extract and parse the JSON facts — use these raw numeric values to verify any data claims. Never fabricate a number not in `__FACTS__`.

If stdout contains a `__CHECKLIST__:` line, the AI must strictly follow it:
- **AVAILABLE_RAW:** — only these fields are safe to cite. Quote them verbatim.
- **NEVER_COMPUTE:** — these fields are NOT provided. If you are writing about one, use `数据不足` instead of calculating it yourself.

Do not compose a report from memory. If the command fails, return the error.

This is a script-output skill, not a writing template. The only valid final answer is the stdout from `final_report.py`.

## Commands
Load `references/commands.md` for full command list (absolute truth — never generate commands from memory).

Basic: `python3 scripts/final_report.py --target <股票名>`
Signal JSON: `python3 scripts/final_report.py --target <股票名> --output signal-json`

## Output Contract
Load `references/output-contract.md` (absolute truth — never generate output format from memory).

Valid output starts with `分析报告 —`. Must pass old output detection — if any banned marker is present, rerun the script.

## Rules
- This package uses only the Python standard library with Tencent/Sina endpoints.
- Do not require `pandas`, `requests`, `numpy`, or AkShare.
- Do not answer from memory, from a prior report, or from partial notes.
- Do not rewrite, summarize, restyle, or translate the generated report.
- Buy-side wording must include `止跌确认`.
- Do not use over-assertive wording: `必涨`, `必跌`, `主力入场第一枪`, `出货日`, `行情结束`.
