# Trader

## Critical Rule
The report is generated by code, not by the agent.

Run `scripts/final_report.py --target <股票名或代码>`, return stdout exactly, and do not compose a report from memory. If the command fails, return the error.

This is a script-output skill, not a writing template. Hermes must not draft, summarize, shorten, translate, reformat, or add follow-up lines after the script output. The only valid final answer is the stdout from `final_report.py`.

## Commands
```bash
python3 scripts/final_report.py --target 南网科技
python3 scripts/final_report.py --target 南网科技 --output signal-json
python3 scripts/validate_output.py /path/to/report.md
python3 scripts/self_check.py
```

`--output signal-json` is for downstream consumption only. The default user-facing output is the validated Markdown report.

## Output Contract
Valid output starts with:

```text
分析报告 —
```

Headings must appear in this order:

```text
✅ 结论概述
🎯 今日交易计划
📏 仓位管理
🧭 简化分析逻辑
⚠️ 风险管理
📌 交易指导卡
👉 一句话
```

Required rules:
- Top block includes `MA5 / MA10 / MA20 / MA30`; use `--` if unavailable.
- Do not show `分析时间` or `规则版本`.
- Use exactly two markdown tables: `今日交易计划` with `角色 / 行动`, and `交易指导卡` with `操作 / 条件`.
- Do not turn `📏 仓位管理`, `🧭 简化分析逻辑`, `⚠️ 风险管理`, or `👉 一句话` into tables.
- Do not use `##/###`, bold headings, blockquotes, bullet lists, or extra disclaimers.
- Do not output intraday execution prices, intraday trading cards, or concrete order instructions.
- Do not output `⏱️ T0 简版`, `T0`, `做T`, `t0-trader`, or `执行价`.
- `🧭 简化分析逻辑` must include `结构：`, `量价：`, `筹码压力：`, `动能：`, and `反向信号：`.
- Buy-side wording must include `止跌确认`; observation zones are not buy points.

## Old Output Detection
If the generated answer contains any of these, it is not valid Trader output and you must rerun the script and return stdout verbatim:

```text
⏱️ T0 简版
T0
做T
t0-trader
执行价
✅ 先给结论
🎯 今日行动
📏 仓位上限
🧭 为什么
⚠️ 如果走势不对
📌 最终行动卡
```

Valid Trader output has exactly two markdown tables. If more than two tables appear, rerun the script.

## Rules
- This package uses only the Python standard library.
- Tencent/Sina live endpoints are the only default data sources.
- Do not require `pandas`, `requests`, `numpy`, or AkShare.
- Do not answer from memory, from a prior report, or from partial notes.
- Do not rewrite, summarize, restyle, or translate the generated report.
- If validation fails after copying, rerun the script and return the script output format.
- Do not call the lightweight pressure estimate `筹码峰｜`; use `筹码压力｜`.
- Do not use over-assertive wording such as `必涨`, `必跌`, `主力入场第一枪`, `出货日`, or `行情结束`.
