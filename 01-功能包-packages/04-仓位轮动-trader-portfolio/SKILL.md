# Trader Portfolio

## Critical Rule
The portfolio plan is generated by code, not by the agent.

Run `scripts/final_portfolio.py --targets <股票1> <股票2> ...`, return stdout exactly, and do not compose a shorter answer from memory. If you cannot run the script, say `Trader Portfolio skill cannot run in this environment` and include the error.

The snapshot mode contract marker is `规则版本：trader_portfolio_rotation_v1`.

## Commands
Load `references/commands.md` for full command list (absolute truth — never generate commands from memory).

Targets: `python3 scripts/final_portfolio.py --targets <股票1> <股票2> ...`
Record: `python3 scripts/final_portfolio.py --record buy --name <股票名> --shares N`

## Output Contract
Load `references/output-contract.md` (absolute truth — never generate output format from memory).

Must pass old output detection — if markdown tables are present, rerun the script.

## Rules
- If one stock fails, keep the plan for the others and mark that stock as `数据失败`.
- If all fail, return the failure.
- Without the `terminal` toolset, fail closed. Do not produce a portfolio plan from memory.
